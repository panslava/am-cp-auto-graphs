Максимальный поток минимальной стоимости

Входной файл:

первая строка
n - число вершин
m - число рёбер
k - ограничение на макс поток (то есть например, мы знаем, что макс поток = 20, а просят найти макс поток минимальной стоимости, которые меньше на 5, тогда k = 15)
вторая строка
s - номер истока
t - номер стока
затем идет m ребер в формате
a b f c
a — откуда
b — куда
f — пропускная способность
c — стоимость транспортировки единицы потока

Выходной файл:
Первая строка — получившийся поток
Список рёбер в формате
a b f
a — откуда
b — куда
f — количество потока по ребру
