Задание

Дан ориентированный граф. Найти компоненты сильной связности.

Входные данные

Первая строка содержит два числа n и m. n - число вершин, m - число ребер.
Следующие m строк содержат описание ребер, по одному в строке. Ребро i описывается двумя натуральными числами bi, ei - началом и концом соответственно.

Ответ

1 строка - количество компонент сильной связности.
2 строка - список номеров компонент для каждой вершины.
